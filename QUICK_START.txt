⚡ GET STARTED IN 30 SECONDS

1. COPY .env TEMPLATE
   cp .env.example .env

2. START BACKEND (Terminal 1)
   cd server
   npm install
   npm start

3. START FRONTEND (Terminal 2 - root directory)
   npm install
   npm run dev

4. OPEN BROWSER
   http://localhost:5173

✅ DONE! You should see emails with AI analysis on the dashboard!

====================================================================

📋 REQUIREMENTS BEFORE YOU START

  ✅ Node.js 20+ installed (check: node --version)
  ✅ npm 10+ installed (check: npm --version)
  ✅ Git installed (optional, for repo management)
  ✅ You're in the project directory

====================================================================

🚀 USING DOCKER (Alternative - Recommended for Production)

1. Make sure Docker is installed and running

2. Create .env file:
   cp .env.example .env

3. Start everything:
   docker compose up --build

4. Open browser:
   http://localhost:8080

✅ The full system will be running!

🤖 ENABLE CLAUDE AI (Optional - For Better Classification)

1. Get Claude API key from https://console.anthropic.com
   
2. Add to .env:
   ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxx

3. Restart server

Now emails will be classified by Claude AI!

====================================================================

📧 USE REAL GMAIL (Optional - To Process Your Inbox)

1. Enable Gmail App Passwords:
   - Go to https://myaccount.google.com
   - Security → App passwords
   - Select "Mail" and "Windows Computer"
   - Copy the 16-character password

2. Update .env:
   EMAIL_MODE=imap
   IMAP_USER=your-email@gmail.com
   IMAP_PASSWORD=xxxx xxxx xxxx xxxx

3. Restart server

Now it will process emails from your Gmail!

====================================================================

🆘 TROUBLESHOOTING

Port 3001 or 5173 already in use?
→ Change AGENT_PORT in .env or kill the process using the port

Dependencies error?
→ rm -r node_modules && npm install

Database error?
→ rm data/emails.db && restart

Server won't start?
→ Check npm install completed: cd server && npm install && cd ..

📞 COMMON COMMANDS

View server logs:
  cd server && npm start

View frontend logs:
  npm run dev

Check if backend is running:
  curl http://localhost:3001/health

Trigger manual email run:
  curl -X POST http://localhost:3001/run

Get all notifications:
  curl http://localhost:3001/notifications

Reset all data (testing):
  curl -X POST http://localhost:3001/reset
